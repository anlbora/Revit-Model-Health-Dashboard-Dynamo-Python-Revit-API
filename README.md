# Revit Model Health Dashboard

A Dynamo + Python + Revit API automation tool that audits Revit model quality and exports an interactive HTML dashboard for BIM coordination and model governance workflows.

---

## Overview

The **Revit Model Health Dashboard** analyzes a Revit model and generates a visual HTML report containing model health indicators, quality checks, quantity summaries, and detailed model inventories.

The goal of the project is to transform raw BIM data into actionable insights that support BIM coordinators, BIM managers, and project teams during model review processes.

---

## Features

### Model Health Score

Calculates an overall model health score based on configurable quality indicators.

The scoring system combines:

* Positive contributions for healthy model conditions
* Weighted deductions for model issues
* Status classification (Excellent / Good / Fair / Needs Attention)

---

### Quality Checks

The dashboard currently evaluates:

#### Published Views Without Templates

Checks only views placed on sheets and identifies those without assigned view templates.

#### Unplaced Views

Detects views that are not placed on sheets.

#### Unplaced Rooms

Identifies rooms that have not been properly placed in the model.

#### CAD Imports

Detects imported CAD files.

#### Revit Warnings

Reports model warnings generated by Revit.

#### Duplicate Element Warnings

Identifies duplicate-related warnings separately.

---
